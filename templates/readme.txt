Guest Author is a WordPress plugin that allows you to add a post author without the need to register them as users in your site.

== Features ==
* ✍️ Behaves as if they were a regular author.
* 👤 Ability to upload your own avatar image.
* 🌍 Ability to link to a custom page for the guest author.
